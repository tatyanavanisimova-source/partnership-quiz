How It All Works — The Big Picture

Think of it like a chain. Each piece does one job. When a visitor takes the quiz, here is what happens step by step:

VisitorOpens the quiz link
→
VercelServes the website
→
QuizThey answer 15 questions
→
QuizThey enter name + email
→
ResendSends emails
→
ResultThey get results. You get lead.

The quiz code lives on GitHub. When you push a change to GitHub, Vercel automatically picks it up and updates the live website within 1–2 minutes. The domain partnershiparchitect.uk (bought on Namecheap) points to Vercel so visitors see a clean URL instead of a long vercel.app address. Resend is the email delivery service — it sends the results email to the quiz taker and a lead notification to you.

https://namecheap.com

You own partnershiparchitect.uk here. Namecheap is like the land registry — it proves you own the domain name and controls where it points. You set up DNS records here so that the domain points to Vercel (website) and Resend (email).

Login: namecheap.com → your account

4. Resend — What Sends the Emails

https://resend.com

Resend is the postal service. When someone completes the quiz, your code tells Resend to send two emails: one to the quiz taker with their results, and one to you with their details. Resend delivers those emails reliably and keeps logs so you can check if they were delivered.

  • Check if an email was sent: Resend → Emails (left sidebar) — shows every email sent, status, and any errors
  • Check domain is verified: Resend → Domains → should show green "Verified"
  • Your API key: Resend → API Keys — this is stored in Vercel, you don't need to touch it
Login: resend.com → account: yourtimeforchanges@gmail.com

What File to Edit for Common Changes

All files live in the folder: C:\Users\tatya\partnership-quiz\app\

app/PartnershipQuiz.tsx
⭐ The main quiz file. Edit this to change: quiz questions, answer options, dimension names, score bands, welcome text, results text, contact form text, brand colours. This is the one you'll touch most often.
app/page.tsx
A tiny file — just loads the quiz. You rarely need to touch this.
app/layout.tsx
Controls the page title and description that appear in Google search results and when you share the link on LinkedIn. Edit this to update the SEO text.
app/api/send-results/route.ts
The email engine. Controls what the emails look like, who they're sent to, and what data they contain. Edit this if you want to change the email format or add/remove information in the notification you receive.

How to Change a Quiz Question (Step by Step)

  1. Open Cursor on your laptop
  2. Click File → Open Folder → select C:\Users\tatya\partnership-quiz
  3. In the file tree on the left, click app → click PartnershipQuiz.tsx
  4. Press Ctrl+F to search — type part of the question text you want to find
  5. Edit the text directly
  6. Press Ctrl+S to save
  7. Open the Terminal in Cursor: View → Terminal
  8. Type: git add . then press Enter
  9. Type: git commit -m "Updated question" then press Enter
  10. Type: git push then press Enter
  11. Wait 1–2 minutes → your live site updates automatically
⚠ Important: Always use the AI chat in Cursor (press Ctrl+L) if you're unsure. Just describe what you want to change in plain English — e.g. "change the second question in dimension 3 to say X instead of Y". The AI will make the edit safely without breaking anything else.

If Something Goes Wrong

Quiz not loading / white screen
Go to Vercel → Deployments. Is the latest one green ("Ready")? If red, click it → Logs to see the error. Copy the error and paste it into Cursor chat (Ctrl+L).
Emails not arriving
1. Check spam folder first. 2. Go to resend.com → Emails — does it show the email as sent or failed? 3. Go to resend.com → Domains — is partnershiparchitect.uk showing as "Verified" (green)? If not, DNS hasn't propagated yet — wait a few more hours.
Domain not working (partnershiparchitect.uk gives an error)
DNS changes can take up to 48 hours to fully propagate worldwide. The quiz still works at partnership-quiz.vercel.app in the meantime. Check Vercel → Domains — does it show a green tick next to partnershiparchitect.uk?
You want to change something but don't know how
Open Cursor → press Ctrl+L → describe what you want in plain English. For example: "I want to change the welcome headline to say X" or "Add a new question to dimension 2". The AI will make the change for you.
